# Day 30: 總結與未來展望 (Summary & Future)

## 0. 恭喜完賽！🎉
恭喜你完成了這趟為期 30 天的機器學習馬拉松！
從最基礎的線性回歸，一路走到深度學習、電腦視覺、自然語言處理，最後甚至親手打造了一個 AI Web App。
這不是終點，而是你成為 AI 工程師的起點。

## 1. 旅程回顧 (The Journey)
讓我們回頭看看這 30 天我們攻克了哪些山頭：

### 第一階段：打地基 (Day 01-05)
*   搞懂了 AI/ML/DL 的關係。
*   學會了 **線性回歸 (Linear Regression)** 預測數值。
*   掌握了 **資料預處理 (Preprocessing)**：缺失值、One-Hot、標準化。

### 第二階段：監督式學習 (Day 06-12)
*   學會了 **分類 (Classification)**：邏輯回歸、KNN、SVM、決策樹。
*   看懂了 **模型評估**：混淆矩陣、ROC/AUC、F1-Score。

### 第三階段：非監督式學習 (Day 13-17)
*   學會了 **分群 (Clustering)**：K-Means、DBSCAN。
*   掌握了 **降維 (Dimensionality Reduction)**：PCA。

### 第四階段：集成學習 (Day 18-22)
*   學會了 **Random Forest** 和 **XGBoost/LightGBM** 這些競賽神器。
*   了解了 Bagging 與 Boosting 的差異。

### 第五階段：深度學習 (Day 23-28)
*   進入了神經網路的世界：**MLP** (全連接層)。
*   攻克了電腦視覺：**CNN** (卷積神經網路)。
*   挑戰了時間序列：**RNN/LSTM**。

### 第六階段：落地應用 (Day 29)
*   使用 **Streamlit** 將模型部署成 Web App。

## 3. 未來學習地圖 (Next Steps)
機器學習的領域浩瀚無垠，接下來你可以往哪裡走？

### 3.1 實戰派：Kaggle 競賽
*   **做什麼**：參加 Kaggle 上的數據科學競賽。
*   **為什麼**：這是練功最快的地方，你可以看到世界頂尖高手的 Code (Kernels)。
*   **推薦**：先從 "Titanic" 或 "House Prices" 這種 Getting Started 比賽開始。

### 3.2 學術派：閱讀論文 (Papers)
*   **做什麼**：關注 ArXiv 上的最新論文。
*   **為什麼**：AI 進步神速，最新的技術 (如 Transformer, Diffusion Models) 都在論文裡。
*   **推薦資源**：[Papers with Code](https://paperswithcode.com/) (論文 + 程式碼，超推！)。

### 3.3 工程派：MLOps
*   **做什麼**：學習如何將模型「規模化」與「自動化」。
*   **關鍵字**：Docker, Kubernetes, MLflow, CI/CD。
*   **目標**：成為能獨當一面的 Machine Learning Engineer。

### 3.4 潮流派：Generative AI (生成式 AI)
*   **做什麼**：學習大型語言模型 (LLM) 與擴散模型 (Diffusion)。
*   **關鍵字**：Transformer, BERT, GPT, Stable Diffusion, LangChain。
*   **目標**：開發像 ChatGPT 這樣的應用。
